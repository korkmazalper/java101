# Merge Sort
## [Proje Konusu] (https://app.patika.dev/courses/veri-yapilari-ve-algoritmalar/merge-sort-proje)
[16,21,11,8,12,22] dizisinin megesort algoritmasi ile cözüm basamaklarini yaziniz.
## Cözüm
 -                 [16,21,11,8,12,22]
 -            [16,21,11]       [8,12,22]
 -           [16,21] [11]     [8,12] [22]
 -            [11,16,21]      [8,12,22]
 -                [8,11,12,16,21,22]

## Time Complexity 
    O(nlogn)
